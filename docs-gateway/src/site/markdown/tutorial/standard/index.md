数据标准
====================

引言
---------------------

数据标准是患者所有健康档案的框架，在平台与第三方应用进行交互的时候，需要依赖数据标准。理解数据标准对正常调用API有非常大的帮助。
数据标准由以下四大核心元素构成：

- [CDA文档]
- [数据集]
- [数据元]
- [字典]

各元素之间的关系如下：

![CDA文档结构](../images/CDA文档结构.png)

平台数据标准是经过简化后的模型，即数据元之间的嵌套关系不再使用。
